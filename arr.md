## Arrays
---------------
* Script for user name and password by using read command password printing on the sreen
* SheBang
 ```
 #!/bin/bash
 read -p "type your name: " name
 read -p "type your password: " pass
 echo -e "\nUser=\nname\nPassword=$pass"
 wq!
 chmod +x script.s 
 ```
